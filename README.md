**Installation guide**

git clone https://github.com/socute727/pyfetch

cd pyfetch

chmod +x ~/pyfetch/installpyfetch

chmod +x ~/pyfetch/dependences

~/pyfetch/dependences

sudo ~/pyfetch/installpyfetch

(you also need to install python3 and python3-pip)
